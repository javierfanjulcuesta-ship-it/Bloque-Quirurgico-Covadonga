# Usuarios de demostración

En modo DEMO el acceso es por **selección de usuario** en la pantalla inicial (sin contraseña). Seleccione uno y pulse **"Entrar en modo demo"**.

| Rol                 | Correo                 | Nombre                   |
|---------------------|------------------------|--------------------------|
| Gestor/Anestesista  | gestor-anestesista@demo | Gestor Anestesista Demo  |
| Gestor              | gestor@demo            | Gestor Demo              |
| Anestesista         | anestesista@demo       | Anestesista Demo         |
| Cirujano            | cirujano@demo          | Cirujano Demo            |
| Endoscopista        | endoscopista@demo      | Endoscopista Demo        |

**Pantallas según rol:**
- **Cirujano** y **Endoscopista**: `/cirujano` (estado del bloque, programar pacientes).
- **Gestor**, **Anestesista** y **Gestor/Anestesista**: `/calendario` (calendario, asignar anestesistas, mensajes, etc.).
